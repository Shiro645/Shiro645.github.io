What is an OS (Operating System) ?

Definition

An operating system (OS) is a fundamental system software that controls, coordinates, and manages the entire operation of a computer or digital device. It connects the hardware (processor, memory, disks, screen, etc.) and the programs (applications) running on it, so that they can function properly without having to manage all the technical details themselves.
