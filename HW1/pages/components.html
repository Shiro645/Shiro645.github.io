Components of an operating system

An operating system is composed of several software elements, each with a specific role. Together, they ensure the computer's proper functioning, security, and performance.

1. The Kernel

The kernel is the heart of the operating system. It is loaded into memory when the computer starts and remains constantly running.

Its main functions are :


  1. processor management (scheduling of tasks)
  2. memory management
  3. the communication with the equipment via the pilots
  4. the management of processes


The kernel operates in privileged mode, which allows it to directly access hardware, unlike applications.

2. The file system

The file system organizes and manages data stored on disks.

It defines how the files are: named, stored, read and modified, protected.
Thanks to it, the user can work with files and folders, without worrying about the actual location of the data on the disk.
Each OS can use different file systems (e.g. NTFS, ext4, APFS).

3. Security

Security is an essential component of the operating system.
It allows controlling access to computer resources in order to avoid abuse or attacks.

This includes in particular:


  • the management of users and permissions
  • the protection of files and processes
  • the isolation between the programs
  • the prevention of unauthorized access

These mechanisms ensure that each user and application only accesses those resources to which they are authorized.

4. The network

The network component allows the computer to communicate with other machines via local networks or the Internet.

He manages: the sending and receiving of data, the network protocols, the connections between remote applications.

Thanks to this component, functions such as web browsing, file sharing or online communications are possible.
