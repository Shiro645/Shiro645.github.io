An operating system performs several essential functions that allow the computer to function efficiently, stably, and securely.

Ressource Management

One of the main roles of an OS is the management of computer hardware resources. It decides which program uses which resource, when and for how long.
This concerns in particular:

  • the CPU (processor): program execution time
  • random access memory (RAM): memory space allocated to each application
  • the inputs/outputs: keyboard, mouse, screen, discs, printers

The OS acts as an arbiter: it prevents one program from monopolizing all resources or interfering with others. This ensures the stability of the system and avoids software conflicts.

Interface between hardware and software

The operating system acts as a bridge between hardware and programs. Applications do not communicate directly with the processor, memory or peripherals: they pass through the OS.


Thanks to this abstraction:
  • the developers do not need to know the technical details of the hardware
  • the same program can run on different machines
  • the material is used in a safer and controlled way.

Without this interface, each application would have to manage the hardware itself, making software much more complex to develop.

Multitasking

A modern operating system allows multitasking, i.e. running several programs at the same time.
In reality: the processor alternates very quickly between tasks, each program has the illusion of running continuously, the OS plans the order and execution time of each process.
This mechanism allows, for example: to listen to music while browsing the Internet, to download a file while another program is running or to use several applications without blocking the system.
